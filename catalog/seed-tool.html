🌱 RepSprout Catalogue Seed Tool

This is a standalone dev page — it is not linked from the app and is not loaded by index.html. It runs the same idempotent seed described in catalog/plantRegistrySeed.js against the real Firebase project. Nothing here connects to gameplay or changes the running app; it only writes to catalog/plants and catalog/collections in Firebase RTDB.

Before running this for the first time, add the catalog rule from database.rules.json to the Firebase Console (Realtime Database → Rules) — same manual step already required for every other rule change in this project, since there is no CLI deploy pipeline here.
