How Double Progression works

  • looks_oneEach working set progresses on its own — one set can be ready to increase while another is still building reps.
  • looks_twoThe maximum rep target comes from the rep range you set for this exercise, never a fixed number.
  • looks_3Reaching the maximum reps (at a comfortable RIR) makes a set ready for more weight.
  • looks_4Moving that set to a heavier weight matures its leaf.
  • looks_5Once every set's leaf has matured, the plant evolves to its next stage and a new cycle begins.
